Connecting to camera...

The system will automatically detect and display health code status (GO/STOP) in the video feed.

How It Works

  • Position the health code within the camera view
  • The system will automatically detect and analyze the health code
  • Green health codes will show "GO" with a green indicator
  • Red or invalid health codes will show "STOP" with a red indicator
  • An audible beep will sound when a health code is detected
